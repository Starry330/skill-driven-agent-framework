# 网格划分指南（静力学）

## 目标
网格策略应服务于位移、应力、反力等关键结果的稳定性，而不是单纯追求单元数量。

## 推荐优先级
1. 先识别危险区域，再谈局部细化。
2. 先保证载荷区、约束区、接触区可被正确分辨。
3. 对圆角根部、孔边、截面突变处优先细化。
4. 对与承载路径无关的小特征优先做几何简化。

## 建议检查项
- 单元长宽比
- 扭曲与 Jacobian
- 局部过密/过疏是否导致刚度畸变
- 接触区两侧网格是否足够兼容
- 连续两轮细化后关键指标变化是否收敛

## 常见风险
- 根部圆角单元过少导致热点应力失真
- 孔边网格过粗导致应力集中估计偏低
- 全局固定约束范围过大掩盖真实柔度
- 把假峰值当成真实最大应力
